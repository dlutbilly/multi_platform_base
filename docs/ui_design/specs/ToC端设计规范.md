# To C 端产品 - UI 设计规范与布局逻辑

## 1. 视觉风格 (Visual Style)
- **风格定位:** 现代、活力、沉浸式、高对比度。
- **色彩系统:** 
  - 主色: 渐变色 (如：#FF4D4F 到 #FF7875) 或 鲜艳的品牌色。
  - 背景: 纯白或极浅灰，配合大面积的图片和视频内容。
- **圆角:** 采用更大的圆角 (12px - 24px)，增加亲和力。
- **阴影:** 柔和的弥散阴影，营造悬浮感和层级感。

## 2. 布局逻辑 (Layout Logic)

### 2.1. PC 端布局 (Top Navigation + Grid)
- **顶部导航 (Header):** 
  - 左侧: 品牌 Logo。
  - 中间: 水平导航菜单 (首页、探索、分类、社区)。
  - 右侧: 搜索框、购物车/通知、用户头像。
- **内容区 (Content):** 
  - 采用通栏 Banner 或 走马灯 (Carousel) 展示核心内容。
  - 下方采用 瀑布流 (Waterfall) 或 网格 (Grid) 布局展示产品/内容卡片。
  - 侧边可放置 推荐位 或 热门榜单。

### 2.2. 移动端布局 (Immersive + Bottom Tab)
- **顶部栏 (Navbar):** 
  - 沉浸式设计，背景可随滚动变色。
  - 包含 搜索入口 和 消息中心。
- **内容区:** 
  - 顶部为 宫格导航 (Grid Menu) 用于快速进入各业务模块。
  - 核心内容以 大图卡片 或 视频流 形式展示。
- **底部栏 (Tabbar):** 
  - 5个图标位，中间图标通常为 核心功能 (如：发布、购买、扫码) 的突出显示。

## 3. 交互特点
- **沉浸式体验:** 减少边框和分割线，利用间距和阴影区分内容。
- **动效丰富:** 包含 页面切换动效、按钮点击反馈、下拉刷新动画等。
- **个性化推荐:** 界面布局支持根据用户偏好动态调整内容权重。
