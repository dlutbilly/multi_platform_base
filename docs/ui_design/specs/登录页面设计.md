# 登录页面 UI 设计方案

## 1. 页面结构设计

### 1.1. PC 端布局 (Desktop)
- **背景:** 采用浅灰色渐变或简约的几何图形背景。
- **登录卡片:** 居中显示，宽度 400px。
  - **顶部:** 项目 Logo + 项目名称。
  - **中部:** 
    - Tab 切换: "账号登录" / "验证码登录"。
    - 输入框: 带有前置图标 (User/Lock/Mobile)。
    - 辅助操作: "记住我" 勾选框 + "忘记密码" 链接。
  - **底部:** 
    - 主按钮: "登录" (Primary Color)。
    - 第三方登录: 微信、QQ、GitHub 图标按钮。
    - 注册入口: "还没有账号？立即注册"。

### 1.2. 移动端布局 (Mobile)
- **背景:** 纯白背景，保持清爽。
- **布局:** 全屏流式布局。
  - **顶部:** 占据 30% 高度的 Logo 展示区。
  - **中部:** 登录表单，左右间距 24px。
  - **底部:** 固定在底部的第三方登录区域。

## 2. 交互流程
1. 用户输入信息 -> 实时校验格式。
2. 点击登录 -> 按钮进入 Loading 状态。
3. 登录成功 -> 渐隐切换至主界面。
4. 登录失败 -> 顶部弹出错误提示。

## 3. 视觉草图 (Mermaid)

```mermaid

    subgraph PC_Login_Card
        Logo[Logo & Title]
        Tabs[Account | Mobile]

    end
```
