# 极简风格 - UI 设计规范与全屏布局逻辑

## 1. 视觉规范 (Visual Specs)
- **核心理念:** 少即是多 (Less is More)，强调内容本身。
- **色彩系统:** 
  - 极简黑白灰: #FFFFFF (背景), #000000 (文字), #F5F5F5 (辅助)。
  - 极少量点缀色: 仅在关键行动按钮使用一个高饱和度颜色。
- **排版:** 
  - 使用大字号、高字间距的无衬线字体。
  - 极大的留白 (Negative Space)，让视觉呼吸。
- **元素:** 
  - 极细线条 (Hairline borders)。
  - 几乎不使用阴影，或仅使用极轻微的线性阴影。

## 2. 全屏布局逻辑 (Full-screen Layout)

### 2.1. 沉浸式首屏 (Hero Section)
- **布局:** 全屏背景（纯色或极简摄影图）。
- **内容:** 居中或极左/极右对齐的简短标语 (Slogan) + 一个主行动按钮。
- **导航:** 隐藏式或极简顶部导航，仅保留 Logo 和 2-3 个核心链接。

### 2.2. 模块化滚动 (Section-based Scrolling)
- **结构:** 每一屏为一个独立的功能模块，采用全屏滚动切换。
- **展示:** 每一屏仅展示一个核心信息点，配以精致的插图或产品实拍图。

### 2.3. 极简导航 (Minimalist Navigation)
- **侧边/顶部:** 采用极细的线条勾勒，或使用汉堡菜单隐藏次要功能。
- **交互:** 鼠标悬停时产生轻微的位移或颜色变化，无复杂动效。

## 3. 交互原则
- **静谧感:** 减少干扰元素，让用户专注于核心内容。
- **直觉化:** 布局极其简单，用户无需思考即可理解操作逻辑。
